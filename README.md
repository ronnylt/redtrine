redtrine
========

Redis-based advanced PHP data structures.

Inspired by `Redback`, a high-level Redis library for Node.JS

Installation
============

 Just run composer install, if you want to run Test Suite use composer install --dev
 
 Executing test suite:

    phpunit
    
    bin/behat
